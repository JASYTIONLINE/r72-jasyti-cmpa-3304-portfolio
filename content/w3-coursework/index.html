CMPA 3304 Week 3

Week 03: CSS Foundations and Layouts.

This week, I shift from architect to interior designer—focusing on how to style, align, and shape the content I’ve built so far.

I’m learning how to use CSS selectors to target elements, control spacing through the Box Model, and organize content with Flexbox to create clean, balancedlayouts. Everything I’ve studied here builds toward my first major project, where I’ll combine these techniques to design a fully styled, professional-looking webpage.

From this page, you can access all Week 03 materials, including the Box Model Practice, and Flexbox Froggy exercises.
