This card demonstrates the CSS box model by using padding, borders, and margins to create space around the content. The box model is a core concept in web design that defines how elements are sized and spaced. I use a <div> as a last resort, preferring semantic HTML5 elements like <article>, <section>, and <header> to give meaning to the structure of the page. If you notice somewhere I used the <div> tag and could have used a more appropriate tag, your feedback is welcome.

If you look at my code, you will notice I use paragraph style comments so I can come back later and study my work and remebner why I did things a certain way.
