CMPA 3304 Week 06

Multi-Page Site Architecture and Navigation Patterns

This week focuses on building multi-page website architecture and experimenting with different navigation patterns. Rather than building a complete site from scratch, this exercise concentrated on the specific skills of linking pages together and styling navigation patterns.

I worked with a template that included root-level pages and nested folders, fixing broken links and CSS paths using relative file paths. I learned how to navigate between root and nested pages using paths like ../ to move up directory levels. I then experimented with at several different navigation styles across the site and on page, customizing colors, spacing, typography, and hover effects to create a navigation laboratory where I could practice and understand how multi-page architecture works.
